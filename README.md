# Show Visit

เว็บตัวอย่างสำหรับเลือกข้อมูลและแสดงผลรายละเอียด

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์
2. เลือกข้อมูลจาก dropdown
3. ดูรายละเอียดที่แสดงด้านล่าง

## ไฟล์สำคัญ

- `index.html` - โครงสร้างหน้าเว็บ
- `styles.css` - สไตล์และรูปแบบ
- `script.js` - ตรรกะการเลือกข้อมูลและการแสดงผล
